Interesting Websites

  • Hardware: Ben Eater has some of the best instructional videos available. In this project, he walks the viewer through building an 8-bit computer on a breadboard: Ben Eater's 8-bit computer.
  • Hardware: construct a computer using a hardware description language. Similar to Ben Eater's project, but with more theory (textbook and slides): From NAND to Tetris.
  • Computer science: some great resources for learning a wide range of computer science principles, from discrete mathematics to compilers and more: Teach Yourself Computer Science.
  • Web development: Mozilla's Learn Web Development guide is one of the best I've seen. It teaches you how the most common web technologies work together, allowing you to see the big picture while learning each one. Plus, it's really easy to refer back to later.
